Custom Zones

Developer guide — add your zone to custom-zones.json with up to 3 JavaScript files.

JSON Schema

Field Type Required Description
id number required Unique integer ID. Use 10000+ for custom zones.
name string required Display name of the zone.
author string optional Author's display name.
authorLink string optional URL linking to the author's profile.
cover string required Absolute URL of the cover image (PNG/JPG).
url string required Absolute URL of the zone HTML file.
jsFiles string[] optional Up to 3 JavaScript file URLs to inject into the zone before it loads.
special string[] optional Tags, e.g. ["custom"].
jsFiles — each URL is injected as a <script src="..."> tag before </body> when the zone is loaded. Maximum of 3 entries.
